Тесты по темам

Present Simple

Используется:

  • привычки
  • факты
  • расписания
(+) S + V(s/es)
(-) S + do/does + not + V
(?) Do/Does + S + V

Present Continuous

Используется:

  • действия, происходящие на данный момент
(+) S + to be + V(ing)
(-) S + to be + not + V(ing)
(?) To be + S + V(ing)

Present Perfect

Используется:

  • действия, завершившиеся к настоящему моменту
  • опыт и результаты
(+) S + have/has + V3
(-) S + have/has + not + V3
(?) Have/Has + S + V3

Present Perfect Continuous

Используется:

  • >действия, начавшиеся в прошлом и продолжающиеся до настоящего
(+) S + have/has + been + V(ing)
(-) S + have/has + not + been + V(ing)
(?) Have/Has + S + been + V(ing)

Past Simple

Используется:

  • действия в прошлом
  • последовательность событий
(+) S + V2
(-) S + did not + V
(?) Did + S + V

Past Continuous

Используется:

  • действия, происходившие в определённый момент в прошлом
(+) S + was/were + V(ing)
(-) S + was/were + not + V(ing)
(?) Was/Were + S + V(ing)

Past Perfect

Используется:

  • действие, которое произошло до другого события в прошлом
(+) S + had + V3
(-) S + had not + V3
(?) Had + S + V3

Past Perfect Continuous

Используется:

  • действие, которое длилось до другого момента в прошлом
(+) S + had + been + V(ing)
(-) S + had not + been + V(ing)
(?) Had + S + been + V(ing)

Future Simple

Используется:

  • действия, которые произойдут в будущем
  • предсказания и обещания
(+) S + will + V
(-) S + will not + V
(?) Will + S + V

Future Continuous

Используется:

  • действия, которые будут происходить в определённый момент в будущем
(+) S + will + be + V(ing)
(-) S + will + not + be + V(ing)
(?) Will + S + be + V(ing)

Future Perfect

Используется:

  • действие, которое завершится до определённого момента в будущем
(+) S + will + have + V3
(-) S + will + not + have + V3
(?) Will + S + have + V3

Future Perfect Continuous

Используется:

  • действие, которое будет длиться до определённого момента в будущем
(+) S + will + have + been + V(ing)
(-) S + will + not + have + been + V(ing)
(?) Will + S + have + been + V(ing)
